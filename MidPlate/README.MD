# Midplate Board Documentation

This board is a power distribution board, including  battery input, reverse polarity protection, power pillars for the chicker board, and a power switch to control power. 

This version is from 2024, published in 2026. The midplate had one failure in the past where a threaded SMT insert had failed due to over-torque, but has only had one such occurrence.
